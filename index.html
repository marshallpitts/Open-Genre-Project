Open Genre Project: Marshall Pitts's Coding Projects

Project 01: Rock Paper Scissors!

Wrote a program that simulates a game of rock, paper, scissors between a human and the of 3 rounds.

Project 02: What is your Zodiac?

Wrote a program that determines a persons astrological zodiac sign based on their birthday

Project 3: A Below Average Winter

Wrote a program that takes in temperatures for 10 days and stores it in an array. Finds the average temperature for those 10 days, and then prints the number of days and the temperatures that were strictly below the average

Project 4: Art Generator

Wrote a program that generates every monochrome image of a given size!

Project 5: Lizard Friend

Wrote a program that allows you to buy and name a lizard!

Project 6: String Helper

Wrote a program that creates a helper class called StringHelper. This class has methods not already built into strings.

Project 7: Peanut Butter and Jelly Sandwich

Practiced object-oriented principles by making two Peanut Butter and Jelly Sandwiches. The program creates two sandwiches based on user input. The sandwich information for both are then printed out and the program determine if the two sandwiches are equal.
